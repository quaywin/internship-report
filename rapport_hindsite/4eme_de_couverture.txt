Participation au principal projet : "Easy Web Content" qui consiste en un éditeur de sites web en-ligne et simplifié pour l'utilisateur qui ne requiert aucune connaissance en informatique. Plus précisément j'ai travaillé sur 2 axes principaux : un éditeur d'images utilisant la technologie flash, un générateur de sites en-ligne (Easy Web Content ne se destinait jusque là qu'à l'edition de sites dejà existants).
L'éditeur d'images utilise principalement les techonologies flash et peut également communiquer avec Easy Web Content via la technologie javascript.
Le générateur de site est un projet très complexe utilisant extensivement la technologie AJAX et javascript.
Un gros travail de prospection, d'analyse et d'étude de faisabilité a été requis pour commencer le developpement de ce projet. L'architecture côté serveur utilise un pattern de type MVC réalisé en utilisant le paradigme Orienté Objet. Ceci permettant une scalabilité accrue.
Le générateur permet de créer très simplement un site web ainsi que ses page par simple glisser-déplacer d'éléments provenant de la palette d'outils. Une fois les éléments positionnés sur la page ils peuvent être modifiés (apparence, position, taille ...) 
Le générateur de page pourra utiliser les addons-créés pour Easy Web Content. Il pourra également disposer de widgets variés tels que réseaux sociaux, forum, blog, formulaires, e-commerce ...
En paralèle à ce projet majeur, j'ai participé au développement de 5 sites internet pour les clients d'Hindsite Interactive. Ceci impliquant un relationnel important avec les clients, la gestion des tâches et priorités, parfois la pression de deadlines, l'organisation du travail, la documentation et l'évalutaion du temps nécéssaire aux réalisations.
